# Laravel Cloud Scale Zero Demo

This example demonstrates a simple PHP web application endpoint, similar to what would run on Laravel Cloud. It simulates a small amount of processing time for each request. The example highlights how such an application benefits from Laravel Cloud's managed hosting, cost-efficiency, and 'instant scale to zero' feature, where resources are consumed only when active.

## Language

`php`

## How to Run

1. Ensure you have PHP installed (version 7.4 or higher recommended).
2. Run the built-in PHP web server from the directory containing `index.php`: `php -S localhost:8000`
3. Open your web browser and navigate to `http://localhost:8000`
